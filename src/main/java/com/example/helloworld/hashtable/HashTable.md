# Hash Table - DSA Preparation

---

## Basics of Hash Table
- `hash(key)` → `idx` in the hash table
- `table[idx] = value`
- `HashMap<key, value>`
- `HashSet<key>`

### Common Operations:
- `put(key, value)`
- `get(key)`
- `remove(key)`
- `containsKey(key)`
- `keySet()`
